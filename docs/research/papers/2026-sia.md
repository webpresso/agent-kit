# SIA: Self Improving AI with Harness & Weight Updates

**Citation:** Hebbar et al., arXiv:2605.27276 [cs.AI], v2 2026-05-28. Paper
CC BY-SA 4.0; code MIT (hexo-ai/sia, PyPI `sia-agent`).

## Mechanism

Frames two disjoint research silos attacking the "humans are the bottleneck"
problem: the **harness-update school** (a meta-agent rewrites a task agent's
scaffold — tools, prompts, retry logic, search procedure — weights fixed) and
the **test-time-training school** (RL pipelines update weights — harness
fixed). SIA unifies them: one self-improving loop that updates both the
harness and the weights of a task-specific agent.

## Results (from repo README, citing the paper)

56.6% gain on LawBench, 91.9% runtime reduction on GPU kernels, 502%
improvement on single-cell RNA denoising over baseline. Built on gpt-oss-120b
with LoRA rank 32.

## What agent-kit takes from it

- The **vocabulary**: "harness-update school" is the school agent-kit's
  roadmap joins. Weight updates are explicitly out of scope for agent-kit
  (we don't own models — see VISION.md boundaries).
- An open-source (MIT) reference implementation of a benchmark-gated
  self-improvement loop worth reading before building the regression gate.
